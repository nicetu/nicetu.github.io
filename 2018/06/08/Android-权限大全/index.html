Android 权限大全

发表于 2018-06-08 | 分类于 Android

总结下 Android 中使用的权限，便于快速查找和使用。 permissiongroup只是一个逻辑分组，作用也只有显示应用程序权限给用户看的。实际在权限的使用中用 permission 。

permission_group

type group_name note
String CALENDAR Used for runtime permissions related to user’s calendar. 用户日历相关的运行时权限。
String CAMERA Used for permissions that are associated with accessing camera or capturing images/video from the device. 访问照相机或从设备捕获图像/视频相关的权限。
String CONTACTS Used for runtime permissions related to contacts and profiles on this device. 设备上的联系人和个人资料相关的运行时权限。
String LOCATION Used for permissions that allow accessing the device location. 允许访问设备位置定位信息的权限。
String MICROPHONE Used for permissions that are associated with accessing microphone audio from the device. 从设备的麦克风访问音频数据的相关权限。
String PHONE Used for permissions that are associated telephony features. 与电话功能相关的权限。
String SENSORS Used for permissions that are associated with accessing camera or capturing images/video from the device. 访问照相机或从设备捕获图像/视频相关的权限。
String SMS Used for runtime permissions related to user’s SMS messages. 用户SMS短消息相关的运行时权限。
String STORAGE Used for runtime permissions related to the shared external storage. 共享外部存储相关的运行时权限。

permission

ps: 使用：”android.permission.name”

type name note
String ACCESS_CHECKIN_PROPERTIES Allows read/write access to the “properties” table in the checkin database, to change values that get uploaded. 允许读写访问在checkin数据库中的“properties”表，修改值并上传。
String ACCESS_COARSE_LOCATION Allows an app to access approximate location. 允许应用程序获取粗略的位置定位。

String ACCESS_NETWORK_STATE Allows applications to access information about networks. 允许应用程序访问有关网络的信息
String ACCESS_NOTIFICATION_POLICY Marker permission for applications that wish to access notification policy. 对希望访问通知策略的应用程序的标记权限。
String ACCESS_WIFI_STATE Allows applications to access information about Wi-Fi networks. 允许应用程序访问有关Wi-Fi网络的信息。

String BATTERY_STATS Allows an application to collect battery statistics 允许应用程序收集电池状态。
String BIND_ACCESSIBILITY_SERVICE Must be required by an AccessibilityService, to ensure that only the system can bind to it. 必须由AccessibilityService来发起请求，确保只有系统可以对它绑定。
String BIND_APPWIDGET Allows an application to tell the AppWidget service which application can access AppWidget’s data. 允许应用程序告诉Appwidget service，其可以访问Appwidget数据。
